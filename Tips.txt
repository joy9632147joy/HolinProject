1.img-fluid 圖片隨著父容器伸縮且不跑版(讓圖片填滿但不跑版)
2.border-box 寬度包含padding/border 動padding不爆版
3.object-fit:cover 搭配w-100 圖片隨著父容器填滿 多餘會剪裁 不跑版
4.d-flex justify(水平y) align(垂直x)
5.flex-wrap 空間不構自動換行 解決flexbox鎖死不換行機制
6.z-index 用於不被覆蓋掉  !!!position非static才有用!!!
7.position: relative; 除了top, left, right, bottom這些z-index也要有他才能設定成功
 -> z-index: 1~10：普通的漂浮元素（如小標籤）。
 -> z-index: 100：下拉選單（Dropdown）、導覽列（Navbar）。
 -> z-index: 1000+：全螢幕的蓋板廣告、彈出視窗（Modal）、警告訊息。
8.object-fit: contain 包含完整的圖 但尺寸不合易有留白
9.ransition: transform 0.3s; 平滑過度 0.3秒浮出來
10.transform: scale(1.05 ); 放大5%


11.display: block; 變成區塊才可控padding/margin
12.overflow: hidden; 左上對左上容易留頭去尾 多餘地方直接隱藏
  -> object-fit:cover 現代做法 自動調到完美位子
13.linear-gradient(...) 線性漸層
14.transition: opacity 3s ease-in-out; 透明度 耗時 慢進入-中加速-慢結束 作互動不要瞬間出現的觀念
15.backquote 自動給文字左右縮排
16.position: fixed 要指定定位(箭頭保持頁面下方等等)要寫他 不然bottom.right都會在靜態中不受控
17.opacity 不透明的
18.:focus 跟 :hover的差異 
  ->:focus 按鈕發光、陰影、準備打字會有顏色等，無障礙
  ->:hover 滑鼠過去有反應，手機或平板會失效因為無游標
19.data-bs-toggle="collapse" 這個按鈕的功能是切換 (Toggle) 摺疊狀態
20.data-bs-target="#navbarTopic" 去控制id為...


21.background-size: cover; 填滿螢幕
22.line-height 行高
23.fst-italic 斜體
24.text-muted 預設字體黑變灰(次要資訊)
25.aspect-ratio 長寬比
26.inset: 0; -> 上下左右邊距皆為0
27.carousel-fade 讓輪播圖由淺入深
28.navbar-expand-lg 大螢幕展開lg以下變漢堡選單
29.fixed-top 跟 sticky-top差異
  -> 前者 通常在襖覽列
  -> 後者 通常在次級內容上

JS-單執行序
1.JS裡系統自動生成classlist 這個屬性裡含有多種方法可切換樣式/動畫 是核心遙控器
2.=> 意思為 輸入->輸出
3.sort -> 判斷回傳值的正副 來決定 新到舊或舊到新
4. ? 三元子運算 -> 條件?(判斷為true的值:判斷為false的值)
5. ==(只比數值) / ===(比型別跟數值)
6.active作用
7.DOMContentLoaded 告訴 JS：等 HTML 長出來後 再執行這些程式碼
 -> load (window.onload) 要等所優資源全部出來(包含CSS 大圖 動畫 等等)
8.parse
9.try{}catch(error){} -> 很好用 防NULL/錯誤機制
10.JSON.stringify() 把JS物件轉成字串(網路傳輸只能轉文字)


11.MIME Type 資料類型標籤
 -> text/html - HTML 網頁
 -> application/json - JSON 資料
 -> image/jpeg - 圖片
 -> text/plain - 純文字
12.async 非同步函式 可以同時執行多件事
13.function(e) -> e為變數 當按鈕被點擊，瀏覽器會自動傳一個「事件物件(在哪點 點甚麼..)」給這個函式
14.preventDefault() 房市預設型為例 ex:重整/form標籤有執行的東西(active:...)
15.trim() 去掉前後空白
16.fetch...await 拿...(資料)過來 要一起使用 
  -> 找資料期間的等待，下一行需要await，不然會跳過此行，直接執行再下一行
17.defer 延遲執行 若要引用JS檔 葉面跑不出來 就記得加他
